ES5 于2009年推出

  • 严格模式
  • Array/Object的一系列方法
    • Array.reduce() / filter() / map() / forEach() /
    • Object.defineProperty() / keys() / freeze() / create() / seal()
      • Object.freeze 冻结对象，使其不可扩展、不可配置、不可修改属性值
        • 适用于常量配置、全局配置；函数式编程中，可确保传入的参数不会被意外修改
      • Object.seal 封装对象，使其不可扩展、不可配置、可以修改属性值
        • 适用于防止被意外扩展，但值可变；保护对象结构
  • 其他方法
    • Date.now() / JSON.parse() / JSON.stringify() / Getter、Setter

ES6（ES2015），与ES5差异最大

  • let和const
  • 箭头函数
  • class
  • Promise
  • `` 模板字符串
  • 函式参数预设值
  • 解构赋构
  • Number.isInteger / isSafeInteger / isNaN
  • module： import / export

ES7（ES2016）

  • 指数运算符 **，等同于Math.pow()
  • Array.includes()

ES8（ES2017）

  • async/await
  • Object方法 Object.values() / entries()
  • padStart() / padEnd()

ES9（ES2018）

  • 非同步迭代 for await … of，以串行方式进行非同步操作
  • Promise.finally()
  • rest属性 …rest

ES10（ES2019）

  • Array.flat()
  • Array.flatMap() 解构一层
  • String.trimStart() / trimEnd()
  • Object.fromEntries()，数组改为对象
  • matchAll
  • BigInt
  • try catch绑定
    • try{} catch {}
